# ColorNames
These are the colors used in Bentos Base theme. These colors will vary between the different themes that are in use. 
The colors are used just like the rest of the theme component. We use colors from theme to controll the color palette within the project 

Example usage and import(using styled-component):
```javascript
import { theming } from "@staccx/bento"

const xxx () => {
  return (
    <ColoredBox/> 
)
}
export default xxx

const ColoredBox = styled.div
background-color: theming.color.secondary
```
