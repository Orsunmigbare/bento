# Animations
This story contains several animations that can be used as a global variable in our styled components.

`Protip:` Remember to set the initial placing of the component in addition to the values from the imported keyframe animation! Sometimes when the code is used in production, the animation can "glitch" and make the overall impression bad.

```javascript
import { AppearIn, BounceIn, BounceOut, Fade, Scale, Slide } from "@staccx/bento"
```
