# formatDate
Format date is used to format date from ISO 8601 or a date object to be displayed in a more readable way in an application.


```javascript
import { formatDate } from "@staccx/bento"
```

## Formatters 
The way to decide how the formatting wil become is by the format string that is passed to formatDate
there are several different ways to display a date string, and they can all be formatted with formatDate
```javascript
const date = "2020-02-12T00:00:00"
formatDate(date, "dd.MM.yy") returns "12.02.20"
formatDate(date, "dd.MMMM.yy") returns "12.februar.20"
formatDate(date, "dd.MM.yyyy") returns "12.02.2020"
formatDate(date, "dd.MMMM.yyyy") returns "12.februar.2020"
```



## Standard formatDate
Takes in an ISO 8601 formatted date and converts it into a more readable date format, with default format.
the code snippet below is the formatted date string.

```javascript
    export const formatted = formatDate("2020-02-12T00:00:00")
```

When converted with default format will look like this:

export const formatted = formatDate("2020-02-12T00:00:00")

  <Story name="Standard">
   <p>{formatted}</p>
  </Story>
  


## Custom format

With the format date function you are also able to add your own format on the returned date string

adding yyyy will return the full year
```javascript
export const year = formatDate("2020-02-12T00:00:00", "yyyy")
```
now returns:

export const year = formatDate("2020-02-12T00:00:00", "yyyy")

  <Story name="Custom format">
   <p>{year}</p>
  </Story>


adding MMMM wil return the full month as a string
```javascript
export const month = formatDate("2020-02-12T00:00:00", "MMMM")
```

now returns:

export const month = formatDate("2020-02-12T00:00:00", "MMMM")

  <Story name="format month">
   <p>{month}</p>
  </Story>
  


HH:mm will return hours and minutes

```javascript
export const month = formatDate("2020-02-12T15:30:00", "HH:mm")
```

now returns:

export const time = formatDate("2020-02-12T15:30:00", "HH:mm")

  <Story name="format time">
   <p>{time}</p>
  </Story>  
  


dd/MMMM/yyyy would then return the whole date with day, month and year
```javascript
export const month = formatDate("2020-02-12T15:30:00", "dd/MMM/yyyy")
```

now returns:

export const full = formatDate("2020-02-12T15:30:00", "dd/MMMM/yyyy")

  <Story name="common">
   <p>{full}</p>
  </Story>  
  

## date object
formatDate can also take in a dateObject and format that, the functionality will be the same as with 
a date string in ISO 8601 format.

```javascript
const date = new Date(1525968057000)
export const dateO = formatDate(date, "dd/MMM/yyyy")
```

now returns:


export const date = new Date(1525968057000)
export const dateO = formatDate(date, "dd/MMMM/yyyy")

  <Story name="date object">
   <p>{dateO}</p>
  </Story>
