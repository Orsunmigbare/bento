# CopyToClipboard
This components is used when you would like the user to copy a text to the clipboard.
The component must have a child component to make it functional.


# import

```javascript
import { CopyToClipboard } from "@staccx/bento"
```

# Types of clipboard
There are variants of the CopyToClipboard.
