# Image
This is the Image component. Remember to use the src prop!


```javascript
import { Image } from "@staccx/bento"
```

There are several ways to display an image with the Image component: round, aspectRatio, size, and children.

## Standard Image

<Preview>
  <Story name="Standard">
    <Image src="http://via.placeholder.com/150x150"/>
  </Story>
</Preview>

## Rounded Image
The rounded image uses the prop round and is a boolean.

<Preview>
  <Story name="Rounded">
    <Image src="http://via.placeholder.com/150x150" round/>
  </Story>
</Preview>

## Aspect ratio
You can use the aspect ratio that takes a String.
