# ItemGroup
The ItemGroup object creates spacing between all its children and wraps the items when there is no room.

```javascript
import { ItemGroup } from "@staccx/bento"
```

## Standard ItemGroup
This is the standard ItemGroup. The spacing is default "tiny".

The spacing prop take up to eight types: macro, tiny, small, medium, mediumPlus, large and huge.

<Preview>
