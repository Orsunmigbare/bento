# Divider
This is the Divider component. The component can be used to structure content such as chunks of text, input-groups or to create a thematic section. 


# import

```javascript
import { Divider } from "@staccx/bento"
```

# Types of Dividers
There are several ways to display the Divider component.

## Standard Divider
The standard Divider uses the height of "1", but it can be modified as desired.
