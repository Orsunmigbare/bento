# NewBadge
Badge are used to add additional information to content

```javascript
import { NewBadge } from "@staccx/bento"
```

## Standard NewBadge
Badge takes a number prop and will display that number in the badge

<Preview>
  <Story name="Standard">
   <NewBadge number={11} />
  </Story>
</Preview>



## Long Badge
Badges are not made for too long numbers

<Preview>
  <Story name="Long Badge">
   <NewBadge number={10000} />
  </Story>
</Preview>

## Badge with children
A badge can also take children elements

<Preview>
  <Story name="Children">
   <NewBadge number={10}>Messages</NewBadge>
  </Story>
</Preview>
